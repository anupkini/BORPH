Supported Platforms

BORPH is implemented as a patch to the Linux kernel (2.6.x), the following implementations are currently available:

  • x86: Intel platform with NetFPGA PCI plugin card
  • ARM: Rhino platform with integrated on-board ARM Cortex processor
  • PowerPC: ROACH platform with integrated on-board PowerPC processor, BEE2 platform with Virtex 2 Pro embedded PowerPC processor
Refer to the Platforms page for more information on these platforms.
